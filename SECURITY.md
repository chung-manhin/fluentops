# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in FluentOps, please report it responsibly.

**Email:** security@fluentops.dev

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment:** within 48 hours
- **Initial assessment:** within 5 business days
- **Fix or mitigation:** depends on severity, targeting 30 days for critical issues

## Scope

This policy covers the FluentOps monorepo (`apps/`, `packages/`, `infra/`). Third-party dependencies are out of scope but will be triaged if reported.

## Disclosure

We follow coordinated disclosure. Please do not open public issues for security vulnerabilities.
